# AfriDevelopers Enterprise React App

A full React + Vite production-ready static website for AfriDevelopers, designed for GitHub Pages and the custom domain `www.afridevelopers.com`.

## Includes
- Premium responsive UI
- High-quality Google fonts: Inter, Manrope, Plus Jakarta Sans
- AfriDevelopers logo
- Full services section
- Full products section: AfriPay, AfriSchool, AfriPOS, Vibendo, AfriPredict, AfriRepair, AfriHR, AfriHospital, AfriHotel, AfriERP, AfriCRM, AfriInventory
- Pricing cards in UGX
- Portfolio and industries sections
- Contact form routed to WhatsApp
- Floating WhatsApp button
- Scroll reveal animations and transitions
- SEO meta tags, robots.txt, sitemap.xml
- GitHub Pages workflow
- CNAME for `www.afridevelopers.com`

## Local run
```bash
npm install
npm run dev
```

## Production build
```bash
npm run build
```

## Deploy
Upload all files to your GitHub repository root, set GitHub Pages source to GitHub Actions, then commit to main.
