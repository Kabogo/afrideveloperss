# AfriDevelopers Premium Multi-Page Web App

Built with HTML5, CSS and JavaScript.

## Pages
- index.html
- about.html
- services.html
- products.html
- pricing.html
- portfolio.html
- contact.html

## Features
- Premium responsive design
- WhatsApp buttons replacing quote buttons
- Pricing tabs in UGX
- AfriDevelopers products: AfriPay, AfriSchool, AfriPOS, Vibendo, AfriPredictor, AfriRepair, AfriHR, AfriHotel, AfriInventory
- Service cards for websites, apps, branding, marketing, systems, consultancy and support
- Contact form that opens WhatsApp with a pre-filled message
- Uses the supplied logo and remote Unsplash image URLs

Open `index.html` in a browser to preview.
