# AfriDevelopers Premium React Website

Premium React + Vite website for AfriDevelopers, designed with high-quality fonts, advanced animations, multi-page routing, services, products, portfolio, pricing and WhatsApp lead generation.

## Features
- React + Vite
- Framer Motion animations
- React Router pages
- CountUp statistics
- High-quality Google Fonts: Sora, Plus Jakarta Sans, Manrope, Inter
- Premium dark UI with gold, cyan and purple accents
- Responsive mobile-first layout
- Floating WhatsApp button
- GitHub Pages ready
- Custom domain CNAME included: www.afridevelopers.com

## Run locally
```bash
npm install
npm run dev
```

## Test production build
```bash
npm run build
npm run preview
```

## Deploy to GitHub Pages
```bash
npm run deploy
```
